# Documents Folder

## Required File:

### CV/Resume:
- **File Name:** `Ali_Raza_CV.pdf`
- **Location:** `assets/documents/Ali_Raza_CV.pdf`
- **Format:** PDF (recommended)
- **Usage:** Download CV button in hero section

## How to Add Your CV:

1. **Prepare your CV:**
   - Save as PDF format
   - Name it: `Ali_Raza_CV.pdf`
   - Recommended size: Under 2MB

2. **Add to folder:**
   - Place the file in this folder: `assets/documents/`
   - Make sure the name matches exactly: `Ali_Raza_CV.pdf`

3. **Test:**
   - Click "DOWNLOAD CV" button on website
   - File should download automatically

## Alternative Formats:

If you want to use a different format or name:

1. Update the link in `index.html`:
   ```html
   <a href="assets/documents/YOUR_FILE_NAME.pdf" download="YOUR_FILE_NAME.pdf" class="btn btn-primary">DOWNLOAD CV</a>
   ```
