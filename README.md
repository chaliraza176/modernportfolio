## 🌟 Overview

Welcome to my professional portfolio. This site is a testament to my journey as a **Computer Science Developer** at COMSATS University and as a **Frontend Engineer** at Ayraxs Technologies Inc. Built with performance and aesthetics in mind, it features a custom-built design system, dynamic interactions, and a showcase of my diverse engineering skills.

### ✨ Key Features

- 🎨 **Dynamic Design System**: Custom-built color picker with 8 presets + unlimited custom colors. Themes persist via `localStorage`.
- 🌓 **Aesthetics First**: Deep black "Midnight" foundation with vibrant, glowing accents for a premium look.
- ⚡ **Performance Optimized**: 100% Google Lighthouse-ready with lazy-loading, asset caching, and minimal dependencies.
- 📱 **Adaptive Layout**: Flawless experience across mobile, tablet, and ultra-wide desktops.
- 🎯 **Interactive Tech Orbit**: An animated technology stack that orbits around the profile, reflecting a "gravity" concept.
- 📄 **Dynamic Content**: Functional typing effects, scroll-triggered animations (Intersection Observer), and seamless smooth scrolling.

## 🚀 Local Development

To run this portfolio locally and test your changes:

1. **Clone the Repo**
   ```bash
   git clone https://github.com/chaliraza176/modernportfolio.git
   cd modernportfolio
   ```

2. **Launch with Live Server**
   - Open in VS Code.
   - Install the "Live Server" extension.
   - Click **Go Live** at the bottom right.

3. **Deploy with Netlify CLI** (Optional)
   ```bash
   npm install -g netlify-cli
   netlify deploy --prod
   ```
